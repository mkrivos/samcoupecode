*****************************************************************                                                              **                      Utilities package                       **                                                              *****************************************************************                                                                                                                                Utilities package is set  of  programs  which  may  be  used  tocompress  files,  archive  programs  etc.  Packge  contains  thefollowing programs:                                                                                                               ARCHIV       - archiver with compression                        UNPAK        - unpack archives created by ARCHIV                IMPLDER      - file/memory packer                               SCREENCOM1   - screen compressor                                LIB          - librarian (with simple compression)              PASS         - simple password garbler                                                                                        Below you will find short description of these programs.                                                                                            ************************                                        *                      *                                        *        ARCHIV        *                                        *                      *                                        ************************                                                                                    To move pointer use cursor keys. Press RETURN to select file.                                                                    F1 - SCAN      -read disk directory                             F2 - LINK      -load selected files to memory                   F3 - COMPRESS  -compress loaded files                           F4 - STORE     -save archive                                    F5 - EXIT                                                       F6 - ALL       -select all files                                                                                               COMPRESS options:                                                                                                               MODE: compression mode                                            1 - shrink (I'm not sure...)                                    2 - implode (I'm not sure too. It's much longer than 1)         3 - ??? (I'really don't know...)                                                                                                  Mode 2 gives you best savings, but is the slowest.          SPEED: compression speed                                          0 - slow, but good compression ratio                            7 - the fastest, but worse compression.                                                                                       SKIP: ??? I don't know. I'm using 0 (default value)...                                                                                             ************************                                        *                      *                                        *         UNPAK        *                                        *                      *                                        ************************                                                                                     To move pointer use cursor keys. Press RETURN to select file.                                                                    F1 - SAVE      -save selected files from archive                F2 - EXIT                                                       F3 - ALL       -select all files                                F4 - NONE      -unselect all files                              F5 - INVERSE   -invert selection                                                                                                                  ************************                                        *                      *                                        *        IMPLODER      *                                        *                      *                                        ************************                                                                                     If you want to compress memory to files you have to load all youwant to compress first and then load IMPLODER and press  F2.  Ifyou want to compress file you have to load IMPLODER press F1 andthen enter file name to load. (Before you press F1/F2  you  haveto press any key first...). Then  you  have  to  enter  (defaultvalues are in brackets) :                                       START  (32768)      -start of block to compress                 LENGTH (32768)      -length of block to compress                CALL   (-)          -call address (simply RETURN for non                             autostarting code)                         NAME   (NONAME.PAK) -file name to save                          MODE   (3)          -compression mode                           SPEED  (0)          -compression speed                          SKIP   (0)          -??? (Sorry I don't know...)                                                                                Compression modes are:                                           1 - shrink                                                      2 - implode                                                     3 - both methods                                                                                                               Speed has the same meaning as for ARCHIV.                                                                                       Note that longer files can  be  compressed  even  more  than  15minutes.                                                                                                                                           ************************                                        *                      *                                        *       SCREENCOM1     *                                        *                      *                                        ************************                                                                                     Load program and then enter screen name to compress and  addressyou want it to be stored at. The default value is 32768 for  thefirst screen or first free  address  after  last  screen.  Aftercompression you may save block of screens or stop  the  program.To add next screen to previously compressed run  program  again.To uncompress screen  use  CALL  with  start  address  (one  youentered or one program displayed before compression).                                                                                              ************************                                        *                      *                                        *        LIB v21       *                                        *                      *                                        ************************                                                                                     LIB is archiver with simple compression. Load it  and  the  menuwill be given. When you select LOAD you have to tell whether youwant compressed files (extension .LCB) or not (extension  .LIB).Then file will be loaded and if it was  compressed  it  will  beuncompressed. When you select SAVE compressed file you  have  towait a moment because archive must be  compressed  before  save.The only problem with this compression is that one  of  routines(I'm not sure is it compress or decompress) has some error   (orerrors) and some files can't be uncompressed properly.                                                                                             ************************                                        *                      *                                        *          PASS        *                                        *                      *                                        ************************                                                                                     PASS is very simple password garbler. After you load it you haveto enter file name to load, then password (up to 31  characters)and after a few moments file name to save. If you want to garblefile run PASS on it. If you want to ungarble it,  run  it  againwith the same password.                                                                                                         *******************************                                                                                                 Sorry that description for first four programs are so short (LIBis quite user friendly, and  PASS  is  very  simple)  but  theseprograms came to me from Slovakia  and  the  only  docs  was  inSlovak.
